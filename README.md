## 野望

### 面白いプロダクトを自分自身が一番楽しんで開発できるエンジニア
エンジニアをやっていて結局いちばん大事なのは作るものの面白さだと考えています。

自分自身が作っているものに対して一番魅力を感じて、自分のできることをすべてやりたいと思えるようなプロダクトの開発・指揮をできるエンジニアになりたいです。

特に面白いと感じるのは、

- 潜在的なニーズにアプローチし、気づいたら新しい常識になっているようなプロダクト
- ユーザーと一緒に育てていけるプロダクト

です。

個人的な趣味として`クリエイターエコノミー`関連のプロダクトにも興味があります。

日々使われる技術は新しくなっていくため、自分が本当に面白いと思うプロダクトの開発に携われるように技術のキャッチアップは欠かさないようにしていきたいです。
